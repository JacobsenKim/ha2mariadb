# ha2mariadb

Homeassistant Sqlite3 to mariaDB merger

```
wget https://github.com/JacobsenKim/ha2mariadb/archive/refs/heads/main.zip
```
```
unzip main.zip
```
```
rm main.zip
```
```
mv ./ha2mariadb-main/ha2mariadb ./ha2mariadb
```
```
cd ./ha2mariadb
```
